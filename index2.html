Client Dashboard Creation - Databox

Overview:


Introduction:
As a project manager, you’ll enhance client oversight by implementing automated scorecards in Databox, delivering weekly KPI updates directly to account managers via email. This process leverages Databox’s real-time tracking, flexible notifications, and performance management features to proactively monitor projects, catch issues early, and boost client satisfaction. The following steps will guide you through creating a standardized scorecard setup and template to streamline this task across all client accounts.


Goal:
Set up scorecards in Databox that will automatically send weekly emails to account managers with key metrics.

Tools:
Databox and access to any data source that should be connected. For data with no data source Google Sheets can be integrated also if data from a non-supported platform can be exported into a CSV or XLSX.

Steps:

  1. Log into Databox: https://auth.databox.com/
  2. Select the client account:

  3. Click on Databoards

  4. Add a new databoard. Most of the time the data sources you will need will already be connected.

  5. Choose “Start Blank”

  6. Click to add metric

  7. Choose a data source


    You will be able to select an existing data source or choose an available integration to connect. Most instances the data sources required will already be connected.



    Once you select a data source, you will see the metrics filter

  8. Type to filter metrics further

  9. Click on the metric to configure it


    Click on ‘Type’


    Select ‘Table’

  10. Resize the table and add additional metics as needed.

  11. As you add metrics, know you can:
  1. Rename the metric.
  2. Break it down my a dimension
  3. Filter the metric by dimension
  4. For instance, adding in event count, you can break the metric down but Event Name, then filter to a specific event. You can then rename the metric something like ‘Contact Form Submissions’ for better context.


  1. For weekly (or other breakdowns) you can edit granulation


    Select the duration of the granulation such as weekly or monthly

  2. Be sure to adjust the default date range, you can make it fixed for a set timeframe or make it a rolling range like last 6 months.


  3. Rename the databoard to make it easy to identify


  4. If you need more space in the report, you can create additional databoards and add them to a loop. This basically turns them into a slideshow.

  5. Schedule the databoard to send snapshot of the report (screenshot).



  6. Name the dashboard, especially useful for Looped databoards.


  7. Optional, upload client logo for easier identification in snapshot


  8. Optional, customize the background color/theme for easier client identification
